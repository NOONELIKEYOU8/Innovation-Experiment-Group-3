# 陈荣坤 - 创新实验个人项目

## 第 14 周 - Flutter Hello World 个性化修改

### 项目说明
基于 Flutter 框架创建的 Hello World 个性化 App，按照第 14 周讲义代码模板完成以下修改：

- 修改 AppBar 标题为个人名称
- 修改页面说明文字
- 添加姓名、学号后四位和小组信息
- 保留按钮与计数器交互功能

### 运行方式
```bash
flutter run
```
支持 Chrome 浏览器、Windows 桌面等平台。

### 修改记录
- AppBar 标题：陈荣坤的创新实验 Flutter 首页
- 页面提示语：Hello Flutter！我已经完成第 14 周入门任务！
- 个人信息：姓名、学号后四位、小组编号
- 按钮文案：完成一次打卡
- 计数含义：今日已完成任务
